# Hipster Holiday

Annual holiday party site at [hipsterholiday.com](https://hipsterholiday.com). Built with Astro + React.

Between events, the site shows a countdown to the next party (always the 2nd Friday of December). Each year gets a unique themed page archived at `/YYYY/`.

## Dev

```bash
npm install
npm run dev
npm run build
```

## Deployment

Push to `main` to deploy via GitHub Pages.
